Multiply prefix by postfix on for each number, ex: 2 in the array has a prefix of 1 and has a postfix (to the right) of 24 so 1* 24 = 24. If at the start of array you can assume prefix is 1 and that the end has a postfix of 1 to be able to retain each value. Furthermore, there is no need to create two new arrays for prefix and postfix, we can do this by first doing a pass from left to right (prefix) and storing the result in the output. Next, we do a pass from the right to left (postfix) and multiply the postfix by the prefix output we stored on the last pass.  ^kCCz8chg
